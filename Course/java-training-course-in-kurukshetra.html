    11. What is the main concept of JAVA?

    The main concept of Java is "write once, run anywhere". This means that once you've written Java code, it can run on any device or operating system that has a Java Runtime Environment (JRE). This is made possible by the Java Virtual Machine (JVM), which interprets Java bytecode and converts it into machine code that the target system can understand. This means Java developers don't have to compile separate versions of their code for different systems. Java also follows the principles of object-oriented programming, including encapsulation, inheritance, and polymorphism.

    12. What is the goal of JAVA?

    The goal of Java is to provide a platform-independent and secure programming language that can be used to create powerful and reliable applications. To achieve this goal, Java was designed with several key features. First, the language is object-oriented, making it easier to design, develop, and maintain programs. Second, Java is platform-independent, meaning that Java code can run on any system that has a Java Virtual Machine (JVM). This allows developers to create cross-platform applications that can run on a wide range of devices. Third, Java provides strong security features to protect both the application and the user.
